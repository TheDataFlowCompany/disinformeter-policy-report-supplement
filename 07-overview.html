How to read this supplement

Crosswalk between the policy report and this online supplement, and a complete figure/table index

Purpose of this page

The policy report is written for a non-technical policy audience and is kept short. This supplement carries the underlying evidence, the full instrument, and the reproducible code. This page is the map between the two: it shows which supplement page expands each section of the report, and it indexes every figure and table so you can jump straight to the one you need.

Report ↔︎ supplement crosswalk

Figure index

Every figure in the report carries a stable F-XX identifier. The same identifiers are used throughout this supplement. The figures themselves are committed, print-ready artefacts (PNG at ≥ 300 dpi plus a vector PDF); each is reproduced on the page listed below, alongside the script that drew it.

Table index

How the code is shown

Throughout the supplement, the script behind each artefact is displayed in a folded, syntax-highlighted block that is read live from the repository and not executed on render — so what you see is exactly what produced the committed figure. Click Code to expand it. Inexpensive aggregate tables are read from CSV and rendered live so you can sort and download them. The heavy upstream estimation (SEM, multilevel Bayesian models) is documented in Reproducibility & methods and on the academic site; it is not re-run here.

Numbering is stable. F- and A-/T- identifiers are locked so that the report, this supplement, and any external reuse all refer to the same artefact.
